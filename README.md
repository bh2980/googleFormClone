# Google Form 클론코딩

개발 인원 : 1명

프로젝트 기간 : 2023.11.18 ~ 2023.11.29

설명 : 구글 폼 클론 코딩 토이 프로젝트입니다.

배포 링크 : https://mygoogleformclone.netlify.app

기능

1. 질문 생성, 삭제, 복제
   - 질문 타입
     - 단답형
     - 장문형
     - 객관식
     - 체크박스
     - 드롭박스
   - 필수 여부
2. 제출폼 미리보기
3. 답변 확인하기
4. 반응형 레이아웃 지원
   - ~ 767px : 모바일
   - 767px ~ 991px : 태블릿
   - 992px ~ : 데스크탑
